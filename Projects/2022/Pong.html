Pong

"Pong is a two-dimensional sports game that simulates table tennis" , But I think everyone knows what pong is. This version of pong has increasing ball speed , win screens and custom art. Most of the (V1) code was not written by me , but all the art is mine.

This was my first ever game development project so it is very small and not that interesting. Most of the code was written by my teachers, since I didnt know any coding at all. The game is made in unity with transform translate. Pong has been updated to version 2.0 on December 15th 2023
