Code Armada: Crash Guardians

"The game is a Tower Defense Game, where the main objective is to survive as many waves as you possibly can! If the enemies do manage to attack your main CPU something bad might happen..." It is a endless tower defence game with a small amount of enemies are towers.

Code Armada: Crash Guardians was my first group project, being made as a project in school. This project took about half a year and was made in a group of 5 students. Personally I made the wave spawning and the printer tower (MMM).
